Setting up the Game

  • Run run_server.bat on Windows or run_server.sh on a UNIX-based system.
  • If the server starts, run run_client.bat

The download directory is synchronized with the repository to ensure it always contains the latest updates.
The list of changes is available in the changelogs tab.
Downloading java is not required as it was included in the project.
For more information, check the info directory in the downloaded game archive.
