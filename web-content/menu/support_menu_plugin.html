Support Panel for 2009scape Client (Unofficial plugin)

The Support is a tool ported and updated from legacy client for administrators or modders to easily spawn items and NPCs within the 2009scape client for testing, debugging, or custom gameplay. This panel provides a convenient, in-game interface for managing spawns without requiring command-line inputs or external scripts.

Features:

Item List:

  • A search bar to select the specific Item ID to spawn (double-click to spawn from the list).
  • The ability to specify a amount of the item to spawn, allowing for bulk item creation.

NPC List:

  • A search bar to select the specific NPC ID to spawn (double-click to spawn from the list).
  • A dropdown or search bar to choose the NPC ID or NPC name to spawn (Double-click to spawn)

Teleport Menu:

  • List with locations (double-click to execute).
  • Quick navigation that allows you to teleport to the most important locations in the game (more than 500 locations on the list).

Conflicts:

Please note that plugin currently is incompatible with:

  • GroundItems
  • KondoKit
